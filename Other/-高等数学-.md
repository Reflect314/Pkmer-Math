# 狭义的高等数学

一般特指国内大学的高等数学课程, 主要内容为微积分. 
初等数学研究对象基本上是不变的量, 而高等数学研究对象是变动的量. 

```base
filters:
  and:
    - file.name.startsWith("--")
    - file.name.endsWith("--")
    - file.path.contains("微积分")
    - file.name != this.file.name
views:
  - type: list
    name: 列表
    order:
      - file.name
    sort:
      - property: chapter
        direction: ASC

```
